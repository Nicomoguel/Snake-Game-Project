#Snake game
 -Description
   .-Interactive and user-friendly design of the classic Chrome snake game, built in Python with the use of Tkinter.
   .-Demonstrates the domain of event-driven programming, real time imput, movement, collision, score and more.

 -Main features
   .-Centered Window geometry not resizable for preventing errors.
   .-Movement controls and collision detection with the borders and snake.
   .-Ability to restart the game when it's over, the user decides when to stop playing.

 -Tech Stack 
   .-Python, Tkinter, GUI Design, Cycles, Object-oriented programming.
